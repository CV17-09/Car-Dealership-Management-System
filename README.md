# Car Dealership Management System (SQL)

A database-driven system designed to manage vehicle inventory, sales transactions, customers, and service records for a car dealership.

## Features
- Vehicle inventory management
- Customer and sales tracking
- Service history tracking
- Automated updates using triggers
- Advanced SQL queries for business insights

## Technologies Used
- SQL (MySQL)
- Stored Procedures
- Triggers
- Views
- Joins and Subqueries

## Database Structure
- Vehicles
- Customers
- Sales
- ServiceRecords
- Employees

## Sample Queries
- Monthly sales reports
- Top-selling vehicles
- Customer service history
- Revenue analysis
